# clap_dotenv

Generate a commented `.env` template from a [`clap::Command`](https://docs.rs/clap).

Tracks [clap-rs/clap#4729](https://github.com/clap-rs/clap/issues/4729). Maintainer feedback preferred **companion crate** (same shape as `clap_complete` / `clap_mangen`) rather than growing core `Command`.

## Usage

```rust
use clap::{Command, Arg, ArgAction};
use clap_dotenv::generate_dotenv;

fn cli() -> Command {
    Command::new("demo")
        .arg(
            Arg::new("port")
                .long("port")
                .env("APP_PORT")
                .help("HTTP listen port")
                .default_value("8080"),
        )
        .arg(
            Arg::new("verbose")
                .long("verbose")
                .env("APP_VERBOSE")
                .action(ArgAction::SetTrue)
                .help("Enable verbose logging"),
        )
}

fn main() {
    print!("{}", generate_dotenv(&cli()));
}
```

Example output:

```dotenv
# Generated by clap_dotenv
#
# HTTP listen port
# Type: string
# APP_PORT=8080

# Enable verbose logging
# Type: bool (true/false / 1/0)
# APP_VERBOSE=
```
